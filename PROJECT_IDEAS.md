# 自作プロジェクトのアイデア

技術スタック（Cloud Run、Cloud Spanner）に寄せた実践的なプロジェクト案です。

## 予約管理API

| 項目 | 内容 |
|-----|------|
| **難易度** | 中級 |
| **技術スタック** | gRPC + Cloud Spanner + Cloud Run |
| **学べること** | Spannerのスキーマ設計、日付範囲検索、インターリーブ、トランザクション |

**主な機能:**
- 部屋の空き状況確認（日付範囲検索）
- 予約の作成・変更・キャンセル
- 予約一覧取得（ページネーション）
- 重複予約の防止（トランザクション）

## 在庫管理API

| 項目 | 内容 |
|-----|------|
| **難易度** | 中級〜上級 |
| **技術スタック** | gRPC + Cloud Spanner + Cloud Run |
| **学べること** | 楽観的ロック、Spannerのミューテーション、整合性の担保 |

**主な機能:**
- 在庫の登録・更新・削除
- 在庫数の増減（同時実行制御）
- 在庫履歴の記録
- 低在庫アラート（Server Streaming）
