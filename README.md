# go-grpc-sample

Go言語とgRPCの学習用リポジトリです。

## プロジェクト一覧

| ディレクトリ | 内容 |
|-------------|------|
| [grpc-tutorial](./grpc-tutorial) | gRPC入門 - 基本的なUnary RPC |

## 技術スタック

- **言語**: Go
- **通信プロトコル**: gRPC / Protocol Buffers
- **対象環境**: Google Cloud (Cloud Run, Cloud Spanner等)

## 始め方

```bash
# リポジトリをクローン
git clone <repository-url>
cd go-grpc-sample

# 各チュートリアルへ移動
cd grpc-tutorial
```

詳細は各ディレクトリのREADMEを参照してください。
